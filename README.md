## 下载

### Docker

```
docker pull apachecn0/airflow-doc-zh
docker run -tid -p <port>:80 apachecn0/airflow-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g airflow-doc-zh
airflow-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
